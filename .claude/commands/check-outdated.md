依存関係の簡易チェックを実行し、サマリーを提供します。

## タスク

以下の分析を実行してください：

```bash
pnpm outdated
```

その後、簡潔なサマリーを提供：

1. **重要な更新**（セキュリティまたはメジャーな破壊的変更）
2. **推奨される更新**（有用な機能を持つマイナーバージョン）
3. **オプションの更新**（パッチバージョン、優先度低）

各カテゴリで以下をリスト：
- パッケージ名
- 現在のバージョン -> 最新バージョン
- 変更の種類（パッチ/マイナー/メジャー）
- メジャーの場合は変更点の簡単なメモ

## クイックアクション

以下の具体的なコマンドを提案：
- すべての安全な（パッチ）依存関係を更新するコマンド
- 強く推奨される特定のパッケージを更新するコマンド

回答は簡潔で実行可能なものにしてください。
