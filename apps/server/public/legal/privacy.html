プライバシーポリシー

最終更新日: 2026年1月15日

My Wishlist運営者（以下「運営者」）は、モバイルアプリケーション「My Wishlist」（以下「本サービス」）におけるユーザーの個人情報およびプライバシーの保護を重要と考え、以下のプライバシーポリシーを定めます。

1. 収集する情報

本サービスでは、以下の情報を収集します。

1.1 自動的に収集される情報

  • Device ID（デバイス識別子）: 端末を識別するために自動生成される UUID。アカウント登録の代わりにデータの紐づけに使用します。
  • 利用ログ: アプリの起動日時、機能の利用状況、エラーログなど。
  • 端末情報: OS の種類・バージョン、アプリのバージョンなど。

1.2 ユーザーが入力する情報

  • ウィッシュリストデータ: ユーザーが作成したウィッシュリストアイテム、カテゴリー、ステップ等の情報。
  • 設定情報: 通知頻度、表示設定などの個人設定。
  • 利用規約同意記録: 利用規約・プライバシーポリシーへの同意日時。

2. 情報の利用目的

収集した情報は、以下の目的で利用します。

  1. 本サービスの提供・運営: ユーザーのデータを保存し、複数端末間でのアクセスを可能にする。
  2. サービスの改善: 利用状況を分析し、機能の改善や新機能の開発に活用する。
  3. 通知の最適化: ユーザーの利用パターンを学習し、適切なタイミングで通知を送信する。
  4. AI 提案の生成: ウィッシュリストの内容に基づいて、ステップや行動の提案を生成する。
  5. カスタマーサポート: お問い合わせへの対応やサポートの提供。
  6. 不正利用の防止: サービスの不正利用を検知・防止する。

3. 情報の保管とセキュリティ

ユーザーのデータを安全に保護するため、以下の対策を講じています。

  • 暗号化: 保存データは AES-256 方式で暗号化しています。
  • 通信の暗号化: すべての API 通信は HTTPS/TLS により暗号化されています。
  • アクセス制御: データへのアクセスは認証された端末からのみ可能です。
  • 定期的な監査: セキュリティ対策の有効性を定期的に確認しています。

ただし、インターネット上での完全なセキュリティを保証することはできません。ユーザーは自己の責任においてサービスを利用してください。

4. 第三者への情報提供

運営者は、以下の場合を除き、ユーザーの同意なく第三者に個人情報を提供することはありません。

  1. 法令に基づく場合
  2. 人の生命、身体または財産の保護のために必要がある場合
  3. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合
  4. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合

5. データの削除

ユーザーは、自身のデータの削除をリクエストすることができます。

データ削除リクエスト方法

連絡先: support@mywishlist.app（仮）

必要な情報:

  • 削除を希望する旨の記載
  • Device ID（アプリの設定画面から確認できます）

対応期間: リクエスト受領後、30日以内に対応いたします。

データ削除後は、ウィッシュリスト、カテゴリー、ステップなどすべてのデータが完全に削除され、復元することはできません。

6. Cookie およびローカルストレージ

本サービスでは、以下の目的で Cookie やローカルストレージを使用する場合があります。

  • セッション管理
  • ユーザー設定の保存
  • アクセス解析（匿名化されたデータ）

ブラウザの設定により Cookie を拒否することも可能ですが、一部の機能が正常に動作しなくなる場合があります。

7. 未成年者のプライバシー

本サービスは、13歳未満のお子様を対象としていません。13歳未満のお子様から個人情報を収集していることが判明した場合、速やかにその情報を削除します。保護者の方で、お子様が当社に個人情報を提供したことに気付いた場合は、上記の連絡先までご連絡ください。

8. プライバシーポリシーの変更

運営者は、必要に応じて本プライバシーポリシーを変更することがあります。重要な変更がある場合は、以下の方法でお知らせします。

  • アプリ内での通知
  • 本ページでの告知（最終更新日の更新）

変更後のポリシーは、本ページに掲載された時点から効力を生じます。定期的に本ページをご確認いただくことをお勧めします。
